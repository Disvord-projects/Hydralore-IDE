<h1>Инструкция по запуску</h1>
Скачайте нужные библиотеки
<br>

```
pip install customtkinter
pip install os
pip install sys
pip install subprocess
```

(у вас они могут быть установлены)

Далее запускаем файл ```hydralore.py```
